## タイポグラフィ

### フォントスタック（CSS実値）
- **見出し優先**: `Klee One, serif`（教科書体に近い柔らかい明朝）
- **見出し汎用**: `Klee One, BIZ UDPMincho, serif`
- **本文**: `Klee One, BIZ UDPMincho, Noto Serif JP, serif`

### 特殊指定
- **FV縦書き**: `writing-mode: vertical-rl`（右→左）
- レタースペーシング: `0.05em` / `0.08em`（見出しの和文余裕）
- カスタムトラッキング変数: `--tracking-tight` / `--tracking-widest`

### 採用判断
SWELL 移管時:
- Klee One が雰囲気の核なので **Google Fonts 経由で必須読み込み**
- BIZ UDPMincho を本文Fallbackに
- Noto Serif JP は最終Fallback（既にWPで使われていれば流用）

## 装飾・空間設計

### 角丸（border-radius）
- `2px` / `4px` — ボタン・入力欄
- `10px` — カード（小）
- `20px` — カード（大）・モーダル

### 余白傾向（PCスクショから視覚観察）
- セクション縦余白: 大きめ（PC幅で200px超のセクションも） — 余白で「落ち着き」を出す設計
- セクション背景は `#f8f8f7` ⇄ `#f5f2ed` の交互で識別

### 共通装飾
- セクション見出しは英字キャプション（`SERVICES` 等）を上に小さく、その下に和文太字
- 英字キャプション下に短い区切り線（オレンジ）

## FV（ファーストビュー）デザイン詳細

- **背景**: 都市夜景写真（暗めトーン・被写体は街並み）
- **縦書きキャッチ**: 写真上に縦書きの白テキスト。複数行（おそらく訴求コピー3行）
- **電話CTA**: 中央下にオレンジボタン `0120-356-424`
- **右下バッジ**: `Made with Manus`（manus.space モック由来 → 本番では削除）
- **ヘッダー**: 上部に営業時間バッジ・電話番号・「無料相談」CTA

## CTAバナー（フッター直前）
- 全幅オレンジ（`#b04d3a` 系統と推定）
- 「お気軽にお問い合わせください」系の訴求 + 電話・フォームボタン

## 訴求軸（OG説明・テキスト断片から）

1. **「話だけでも構わない」** — 敷居の低さ
2. **「弁護士特約店」** — 信頼性
3. **「地域密着60年」** — 実績・歴史
4. **「会社でも、個人であれ」** — 法人個人両対応

→ 探偵社サイトの定番訴求と一致。彩いろ案件でも採用すべき軸。

## SWELL 移管時の実装メモ

### 必須実装
- [ ] Google Fonts (Klee One/BIZ UDPMincho/Noto Serif JP) を `wp_head` で `preconnect` + 読み込み（子テーマfunctions.phpか、SWELLのカスタムCSS）
- [ ] FV `writing-mode: vertical-rl` を子テーマCSSで実装（SWELLのフルワイドブロック + カスタムCSSクラス）
- [ ] セクション背景の交互色（`#f8f8f7`/`#f5f2ed`）をSWELLのセクションブロックで再現
- [ ] 英字キャプション付き見出しスタイル（h2の前に英字span）— SWELLの見出しブロックに装飾追加 or カスタムブロックパターン
- [ ] CTAボタンは朱赤系（`#b04d3a`）
- [ ] 角丸トークン: 2px / 4px / 10px / 20px の4段階で統一

### 仮実装で先方確認が必要なもの
- [ ] **配色確定**（藍紺×朱赤系 or 深緑×金茶 のどちらか）
- [ ] FV写真の素材（探偵社らしい夜景 or 自社撮影 or 既存サイトの素材流用）
- [ ] 「弁護士特約店」「地域密着●年」訴求が彩いろ案件でも使えるか

### 採用しない要素
- `Made with Manus` バッジ
- バンドル内に存在する未使用バリエーション（オレンジ／深緑等）
